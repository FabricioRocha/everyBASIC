====== CMD ======

Apparently, allows the user to implement a command for the BASIC REPL environment

**Implemented by:** [[:basics:CBM:]], [[:basics:trs80:level3]], [[:basics:ABasiC:]]

**With variations:** [[:basics:trs80:trsdisk]] (controls interrupts, debugger and communication with TRSDOS), [[:basics:MSX:]] (documented but not implemented)
