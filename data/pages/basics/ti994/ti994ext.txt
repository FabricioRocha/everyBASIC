Released by Texas Instruments in 1981 as a cartridge for the TI99/4, this extension to its original BASIC brought new commands and changed functionality of some already implemented ones.

===== Where it is/was used =====
/* TO DO: Machines and emulators which ran this BASIC. */
  * TI 99/4, TI 99/4A

===== Noteworthy characteristics =====
/* TO DO: a list or prose text about this BASIC's features, quirks, Easter eggs, tricks, singularities... */

  * With the extension, [[.:|TI-BASIC]] could use more than one statement per program line, and also in direct mode.
  * A REDO functionality was implemented to a function key
  * [[:keywords:IF]] could do more than branching after [[:keywords:THEN]], and [[:keywords:ELSE]] was implemented
  * [[:keywords:USING]] implemented for [[:keywords:PRINT]] and [[:keywords:DISPLAY]]
